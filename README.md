# 📈 Stock Market Time Series Analysis

## 📌 Overview
Performed Time Series Analysis on historical stock market data to identify trends, check stationarity, analyze seasonality, and forecast future stock prices.

## 🛠️ Tools Used
- Python
- Pandas
- Statsmodels
- Matplotlib

## 🔍 Analysis Performed
- Data Preprocessing
- ADF Stationarity Test
- Moving Average Analysis
- Seasonal Decomposition
- ARIMA Forecasting

## 📊 Key Findings
- Time series is non-stationary.
- Stock prices show a clear trend over time.
- Moving averages help smooth market fluctuations.
- ARIMA model was used for future forecasting.

## 📈 Deliverables
- TSA Notebook
- Forecast Plot
- Insights Report

## 🎯 Outcome
Gained hands-on experience in Time Series Analysis, forecasting techniques, and financial data analytics.
